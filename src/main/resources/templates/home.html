About This Application

This Student Marks Management System is built using Spring Boot with XML-based configuration instead of annotations, demonstrating the traditional approach to Spring configuration.

The application features:

  • XML-based Spring configuration
  • JPA entity mappings using XML
  • Thymeleaf templates for the frontend
  • Bootstrap for responsive UI
  • Complete CRUD operations for all entities
  • Special reports for top rankers
